## 🚀 Project Overview
1. Purpose: Understand user interactions and sentiments on social media.
2. Importance: Improves user engagement and platform growth.
3. Challenge: Sentiment classification is complex due to multiple influencing factors.

## 📉 Dataset Description
562 labeled examples.
8 features: Text, Sentiment, Timestamp, User, Platform, Hashtags, Retweets, Likes, Country.
3 Sentiment classes: Positive (45%), Negative (30%), Neutral (25%).
Data is balanced.

## ⭐️ Feature and Label Manipulation
- Platform column cleaned (removed spaces).
- Sentiment column encoded to numeric values.
- Importance of cleaning: ensures accurate analysis and visualizations.

## ⭐️ Data Visualization Findings
Likes and Retweets increase yearly.
No clear linear relationship between Likes/Retweets and Sentiments.
Significant class overlap supports the need for classification algorithms.
